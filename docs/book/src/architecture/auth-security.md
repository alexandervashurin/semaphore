# Аутентификация и безопасность

> JWT, LDAP, OIDC, TOTP, RBAC — архитектура безопасности Velum
>
> 📖 См. также: [Обзор системы](./system-overview.md), [Схема базы данных](./database-schema.md), [Конфигурация](../getting-started/configuration.md)

---

## Методы аутентификации

| Метод | Описание | Настройка |
|-------|----------|-----------|
| **JWT (локальный)** | Логин/пароль с JWT-токенами | `VELUM_JWT_SECRET` |
| **LDAP** | Корпоративный каталог | `VELUM_LDAP_*` |
| **OIDC** | OAuth2 провайдеры (Google, GitHub и др.) | `VELUM_OIDC_*` |
| **TOTP** | Двухфакторная аутентификация | Включается в профиле |

## RBAC (Ролевой контроль доступа)

| Роль | Права |
|------|-------|
| **Администратор** | Полный доступ ко всему |
| **Пользователь проекта** | Доступ к конкретному проекту |
| **Только просмотр** | Read-only доступ |

## Безопасность

- JWT-токены с настраиваемым сроком жизни
- Шифрование секретов в базе данных
- Поддержка HTTPS
- CSP-заголовки и защита от XSS
