# Схема базы данных

> 📖 См. также: [Обзор системы](./system-overview.md), [Аутентификация и безопасность](./auth-security.md), [Выполнение задач](./task-execution.md)

---

## Основные таблицы

| Таблица | Описание |
|---------|----------|
| `users` | Пользователи (администраторы, обычные пользователи) |
| `projects` | Проекты — контейнеры для шаблонов и задач |
| `templates` | Шаблоны задач (плейбуки, скрипты) |
| `tasks` | Запуски задач (история выполнений) |
| `task_output` | Логи вывода задач |
| `repositories` | Git-репозитории для playbook |
| `inventory` | Инвентаризация хостов |
| `access_keys` | Ключи доступа (SSH, пароли) |
| `environments` | Переменные окружения |
| `schedules` | Расписания автоматического запуска |

## Миграции

Миграции хранятся в `db/migrations/` и применяются автоматически при запуске.
